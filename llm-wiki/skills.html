MyBatis-Plus 开发辅助 Skill — 完全本地自包含，离线可用

这是什么

给 AI 编程助手用的 MyBatis-Plus 开发 Skill。3.5.x 全线适用，3.4.x 大部分兼容。完全本地自包含，不依赖外部文档站点，离线可用。11 篇本地参考覆盖从依赖配置到事务管理、XML Mapper 编写的全链路，附带 23 条反模式纠偏、输出前 7 项自检清单和多场景交叉路由指引。

适用场景

适用于：项目已集成 MyBatis-Plus（pom.xml 含 mybatis-plus-* 依赖），涉及 CRUD、条件构造、分页、逻辑删除、自动填充、乐观锁、枚举映射、XML Mapper、事务管理等日常开发任务。

不适用于：JPA / Hibernate、数据库表结构设计 / DDL、纯 SQL 性能调优。

部分适用：纯 MyBatis 原生项目（无 MP 依赖）——仅 XML Mapper 编写和事务管理章节可参考。

安装

SkillHub 安装推荐

复制以下提示词，粘贴给你的 AI 编程助手即可自动安装：

请根据 https://skillhub.cn/install/skillhub.md，安装 mybatis-plus-dev。

手动下载 zip备选

直接下载安装包，解压后放入 skills 目录：

curl -L https://baixuanzhu.github.io/mybatis-plus-doc/skills/mybatis-plus-dev.zip -o mybatis-plus-dev.zip

⬇ 下载 mybatis-plus-dev.zip

解压后将 mybatis-plus-dev/ 文件夹放入你所用 AI 编程助手的技能目录即可使用。
